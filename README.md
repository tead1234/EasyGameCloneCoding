# EasyGameCloneCoding

자바스크립트 실력 확인 겸 복습용으로 Wordle이란 게임을 만들어보자
## Wordle이란
랜덤 단어를 맞추는 게임으로
임의의 영어소문자단어를 입력후 제출버튼을 눌렀을때
글자와 자리위치까지 맞으면 회색이
글자만 맞추면 노랑색
다 틀리면 아무 반응도 일어나지 않는 게임이다.
위 방식으로 오늘의 단어를 맞춰보자!

> 부족한 점 (실시간 업데이트)
1. 정답을 다 맞췄을때 폭죽 이벤트가 터졌으면 좋겠다
2. 재시작 버튼만들기
3. 제출을 눌렀을때 틀렸을때만 새로운 입력칸이 나오게 만들어보자
